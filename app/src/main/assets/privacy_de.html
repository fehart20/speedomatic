Datenschutzerklärung

Dieser Speedtest-Server ist mit aktivierter Telemetrie konfiguriert.

Welche Daten werden gesammelt?

Am Ende des Tests werden die folgenden Daten gesammelt und gespeichert:

  • Test-ID
  • Zeitpunkt des Tests
  • Test Ergebnisse (Download- und Upload-Geschwindigkeit, Ping und Jitter)
  • IP Adresse
  • ISP-Informationen
  • Ungefährer Standort (abgeleitet von der IP-Adresse, nicht von Standortinformationen Ihres Gerätes)
  • Verwendeter Browser und dessen Gebietsschema
  • Testprotokoll (enthält keine persönlichen Informationen)

Wie werden die Daten verwendet?

Über diesen Dienst gesammelte Daten werden verwendet, um ...

  • ... das Teilen von Testergebnissen zuzulassen (Nutzbares Bild für Foren etc.)
  • ... den Ihnen angebotenen Service zu verbessern (zum Beispiel um Probleme auf unserer Seite zu erkennen)
Es werden keine persönlichen Daten an Dritte weitergegeben.

Ihre Zustimmung

Mit dem Start des Tests stimmen Sie den Bestimmungen dieser Datenschutzerklärung zu.

Datenentfernung

Wenn Sie möchten, dass Ihre Informationen gelöscht werden, müssen Sie entweder die ID des Tests oder Ihre IP-Adresse angeben. Nur so können wir Ihre Daten identifizieren. Ohne diese Informationen können wir Ihrer Anfrage nicht nachkommen.

Kontaktieren Sie diese E-Mail-Adresse für Löschanfragen: datenschutz@fhcom.de.
