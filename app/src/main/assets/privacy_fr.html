Protection des données

Ce serveur de test de vitesse est configuré avec la télémétrie activée.

Quelles données sont collectées?

À la fin du test, les données suivantes sont collectées et stockées:

  • ID de test
  • Heure du test
  • Résultats des tests (Download et Upload, Ping et Jitter)
  • Adresse IP
  • Informations sur le ISP
  • Emplacement approximatif (dérivé de l'adresse IP et non de vos informations de localisation Dispositif)
  • Navigateur utilisé et ses paramètres régionaux
  • Protocole de test (ne contient aucune information personnelle)

Comment les données sont-elles utilisées?

Les données collectées via ce service sont utilisées pour ...

  • ... permettre le partage des résultats des tests (image utilisable pour les forums etc.)
  • ... améliorer le service qui vous est proposé (par exemple pour résoudre des problèmes de notre côté reconnaître)
Aucune donnée personnelle ne sera transmise à des tiers.

Votre permission

En démarrant le test, vous acceptez les dispositions de cette déclaration de protection des données.

Suppression de données

Si vous souhaitez que vos informations soient effacées, vous aurez soit besoin de l'ID de test ou entrez votre adresse IP. C'est la seule façon dont nous pouvons identifier vos données. Sans ces Nous ne pouvons pas répondre à votre demande d'informations.

Contactez cette adresse e-mail pour les demandes de suppression: datenschutz@fhcom.de.
